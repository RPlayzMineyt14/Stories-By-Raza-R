# 🌻 Stories By Raza R.

> **Official website for my stories and creative writing.**

## 📖 About

**Stories By Raza R.** is a personal author website created to showcase original stories in a clean, elegant and book-inspired design.

The first featured release is:

### ✨ Everything Happens For A Reason

A story inspired by lived experiences, exploring destiny, hope, choices and the meaning hidden behind every chapter of life.

---

## ✨ Features

- 📖 Elegant book-inspired design
- 🌻 Botanical sunflower theme
- 📄 Read Story & Download PDF
- 📱 Responsive layout
- 🎨 Smooth animations
- 📚 Featured Story section
- ⬆️ Back to Top button
- 📊 Reading progress bar
- 💻 Built using pure HTML, CSS & JavaScript

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)
